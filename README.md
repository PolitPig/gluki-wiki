## 📖 About the Wiki

This repository contains only the compiled static player wiki for the **Gluki** plugin. The plugin itself is a modular, high-performance modular Minecraft Paper/Folia plugin designed for economy and roleplay servers (version 1.21.x).

The wiki covers:
- **🌿 Farming**: Seeking wild seeds, growth conditions (podzol, water, UV-lamps), and biome quality multipliers.
- **🧪 Chemistry**: Dryer, Extractor, and Concentrator operations.
- **⚙️ Purity System**: Understanding product chemical purity (0-100%) and avoiding bad trips.
- **⚠️ Hallucinations & Risks**: Glitch levels, cooldowns, overdoses, and addiction.
- **🛠️ Recipes**: Production chains and recipes with clean Minecraft styling.

## 🛠️ Development & Deployment

This wiki is a fully static website built with pure HTML and custom CSS. It is responsive, highly optimized, and uses modern typography (Instrument Serif, Plus Jakarta Sans, JetBrains Mono).

To run locally:
1. Clone the repository.
2. Open `index.html` in your browser.

All edits to the wiki should be made directly in the corresponding HTML files (e.g. `index.html`, `guide/*.html`, etc.). Changes pushed to the `main` branch are automatically deployed to GitHub Pages.
